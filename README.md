# ACE_JOB - 腾讯 ACE 反作弊进程优化工具

本仓库提供一个简单的 Windows 批处理脚本，用于限制腾讯 ACE（SGuard64.exe 和 SGuardSvc64.exe）进程的 CPU 和优先级，减少游戏时的硬盘狂读、卡顿和硬盘损耗。

适用于《三角洲行动》、《和平精英》、《逆战》、《CF》、《CS:GO》等使用腾讯 ACE 反作弊的游戏。

## 文件说明

- **ACE_JOB.bat**（推荐使用）  
  批处理脚本，一键限制 ACE 进程优先级为“低”并绑定到最后一个 CPU 核心。  
  支持 Windows 10 / Windows 11。

- **ACE_JOB.ps1**  
  PowerShell 版本脚本，功能相同（部分电脑可能需要调整执行策略）。

## 使用方法

1. 下载本仓库中的 `ACE_JOB.exe`（推荐）。
2. 启动需要优化的腾讯游戏（确保 ACE 进程已运行）。
3. 右键 `ACE_JOB.exe` → **以管理员身份运行**。
4. 脚本会自动检测并限制 SGuard64.exe 和 SGuardSvc64.exe 进程。
5. 游戏重启后需重新运行一次脚本。

## 注意事项

- 仅调整进程优先级和 CPU 亲和性，完全合规，不修改任何游戏文件。
- 每次游戏重启后需重新运行脚本（设置不会永久保存）。
- 如遇权限问题，请确保以管理员身份运行。
